API Pública

Todos los endpoints son de acceso público sin autenticación. Puedes consumirlos desde cualquier aplicación o script.

GET /api/v1/registry

Estructura JSON cruda para auditorías externas.

curl https://ve-emergency-map.vercel.app/api/v1/registry
GET /api/v1/registry/export

Descarga cruda CSV de contingencia local.

curl https://ve-emergency-map.vercel.app/api/v1/registry/export
GET /api/v1/trends/normalized

Diccionario de hashtags de evasión algorítmica.

curl https://ve-emergency-map.vercel.app/api/v1/trends/normalized
